# KAMI



Debug flags:

| Flag              | Type    | Default Value | Description                                                  |
| ----------------- | ------- | ------------- | ------------------------------------------------------------ |
| kami.debug        | boolean | false         | enable debug logging                                         |
| kami.debug.gltext | boolean | false         | enable debug logging for the [gltext](https://github.com/damien-lloyd/gltext) library |
